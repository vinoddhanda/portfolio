## Overview

91Wheels.com is one of India's fastest-growing automotive media and marketplace platforms — covering cars, bikes, and scooters for millions of monthly users across India.

As **Sr. Design Manager**, I own the complete product experience end-to-end — UX strategy, Figma design systems, Next.js frontend development, Core Web Vitals optimization, and GA4-driven product decisions. No handoffs. No waiting. One owner.

## The Challenge

When I joined in 2020, the platform needed a complete frontend rebuild — performant, scalable, and conversion-focused. The challenge: deliver team-level output with lean resources, while continuously shipping new features and optimizations.

## What I Built

### Complete Frontend — Next.js + Tailwind CSS
Built the entire frontend from scratch using Next.js and Tailwind CSS:
- Vehicle listing pages with advanced search and filter (make, model, price, fuel type, transmission)
- Individual vehicle detail pages with specs, image galleries, and enquiry forms
- Mobile-first responsive navigation and layouts
- Bikes and scooters sections with dedicated UX flows
- Editorial content pages and review templates

### Lead Conversion — 3.5% to 7%
Identified drop-off points in the lead generation flow using **GA4 behavioral data**. Redesigned the enquiry form UX in Figma — simplified fields, reduced friction, improved mobile layout. Shipped the fix in Next.js.

**Result: Lead conversion doubled from 3.5% to 7%** — without any paid traffic increase.

### Core Web Vitals — PageSpeed 90+
Optimized frontend performance across all key pages:
- Image optimization and lazy loading
- Code splitting and bundle size reduction
- Server-side rendering for SEO-critical pages
- Achieved PageSpeed 90+ scores across desktop and mobile

### AI-Native Workflow
Over the last 2 years, rebuilt the entire development and design workflow around AI tools:
- **Claude Code** — primary frontend development environment
- **Cursor** — AI-assisted code review and refactoring
- **Figma AI** — rapid wireframing and design iteration

Work that previously required a 3-4 person team, now delivered solo — faster iteration, lower cost, higher output quality.

### Analytics Infrastructure
Set up complete GA4 + Google Search Console tracking:
- Conversion event tracking across all key user flows
- Funnel analysis for lead generation optimization
- SEO performance monitoring and Core Web Vitals tracking
- Data-driven product decisions — identifying issues before they become complaints
